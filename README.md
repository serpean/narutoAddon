# NarutoAddon

Webextension for vernaruto.net

## Requeriments

- npm 6.7.0
- Mozilla Firefox 65.0

## Install for develop

> npm install
> npm start

If you want to reload webpack in each change, open other terminal and run:
> npm run build:watch


## Build to install

> npm run build:production
